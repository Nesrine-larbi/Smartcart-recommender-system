##  Overview
SmartCart is a data-driven e-commerce recommendation system that suggests products to users based on their behavior and identifies purchasing patterns using machine learning techniques.

This project combines:
- **Collaborative Filtering** to generate personalized recommendations
- **Association Rule Mining (Apriori)** to discover frequent product combinations


## 📂 Dataset
- `ecommerce_user_data.csv` – user interactions (ratings, products, timestamps)
- `product_details.csv` – product metadata (name, category)
